## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`SERVICE_NAME`

`MONGO_URL_DEV`

`MONGO_URL_PROD`

`SECRET_TOKEN`


## Installation

Install my-project with npm

```bash
  cd africa
  npm Install
  nodemon start
```
